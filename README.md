# PyTorch example


## dependencies

- python 3.8

## setup venv

```shell script
python3 -m venv .venv
. .venv/bin/activate
```

## install package

```shell script
make install-package
```

## Run

```shell script
python main.py
```

## update package

when you add some package, update requirements.txt.

```shell script
make save-package
```
